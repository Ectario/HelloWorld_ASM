# HelloWorld_ASM
A simple hello world in assembly.

## Description
Print a certain number of times (which one can choose) "Hello, world". 

Used to learn loop and printing (without C library).

## Configuration used 
- Distribution : Linux
- Processor : Intel
- Architecture : 64 Bits
- Compiler : NASM
- Linker : ld
- Debugger used : gdb-peda
